多文档摘要总结[TODO]

路漫漫其修远兮。

1.任务定义

1.1 定义

从包含多份文档的文档集合中生成能够概括这些文档中心内容的摘要。

1.2 与单文档摘要的对比

（1）难度更大：对于一个文档集合来说，文档越多，其包含的主题、噪声也越多，因此提取摘要的难度也越大；多文档摘要的输入众多，可能包含上百篇文章输入，因此直接使用端到端的模型去训练并不现实。
（2）数据稀缺：相比于单文档，多文档摘要的数据集十分稀缺，在18年之前只有DUC/TAC发布的多文档摘要数据集，每期也只有百余篇。直到2018年才提出了第一个大规模的多文档摘要数据集WikiSum。

2.数据集

（1）主流数据集

  • DUC/TAC 每期只有百余篇
  • 2018 ICLR WikiSum
  • 2019 ACL Multi-News

（2）其它数据集

  • 2020 LREC GameWikiSum

3.现有工作总结

多文档摘要现在的处理方式主要分为三种：抽取式、生成式、抽取生成相结合。

3.1 抽取式

在2018年之前，由于当时没有大规模的训练数据，所以大部分工作所采用的思路，对多文档中的文章和段落进行排序。

3.1.1 基于RNNs

2015 AAAI: Ranking with Recursive Neural Networks and Its Application to Multi-Document Summarization

3.1.2 基于GCN

效果比较依赖于如何计算边的权重。
2017 CoNLL：Graph-based Neural Multi-Document Summarization
2020 ACL：Heterogeneous Graph Neural Networks for Extractive Document Summarization

3.2 生成式

3.2.0 最初的梦想

把多个文档拼接形成长句作为seq2seq的输入，并未考虑多个文档之间的关系，也是两大数据集（WikiSum与Multi-News）被提出的工作。

2018 ICLR：Generating Wikipedia by Summarizing Long Sequences
2019 ACL：Multi-News: A Large-Scale Multi-Document Summarization Dataset and Abstractive Hierarchical Model

3.2.1 微调模型

2018 arXiv：Towards a Neural Network Approach to Abstractive Multi-Document Summarization（微调模型，先在单文档数据集上搞再在多文档数据集上微调）
2018 EMNLP：Adapting the Neural Encoder-Decoder Framework from Single to Multi-Document Summarization

3.2.2 基于句子融合

2015 ACL：Abstractive Multi-Document Summarization via Phrase Selection and Merging

3.2.3 基于重建目标的无监督模型

2016 COLING：An Unsupervised Multi-Document Summarization Framework Based on Neural Document Model
2019 ICML：MeanSum: A Neural Model for Unsupervised Multi-document Abstractive Summarization

3.2.4 考虑文档之间的关系

2019 ACL：Hierarchical Transformers for Multi-Document Summarization

3.3 抽取生成混合

（1）先抽取再生成
2018 ACL：Fast Abstractive Summarization with Reinforce-Selected Sentence Rewriting
（2）将抽取摘要拼接到源文档后
2019 arXiv：On Extractive and Abstractive Neural Document Summarization with Transformer Language Models
（3）抽取生成联合训练让模型更好习得不同粒度的表示
2020 ACL：Multi-Granularity Interaction Network for Extractive and Abstractive Multi-Document Summarization
