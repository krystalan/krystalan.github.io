PyTorch笔记（一）

这一系列文章，主要记录一下自己学习PyTorch的笔记，并且存放一些资料。

1. Tensor

It starts with a tensor

1.1 为什么要使用Tensor

基本上所有的教程，不论你去看文档还是中文教程、英文教程或者说视频，一开始肯定会讲解Tensor，因为Tensor是一个非常重要的概念，是PyTorch当中非常底层的一个东西，也是PyTorch之所以高效的原因所在。首先我们需要知道为什么我们不去用python中自带的list而是用torch.tensor，这个问题其实也比较简单，因为在list当中，每个list的元素其实在内存空间当中并不是连续分布的，而torch.tensor中的每一个元素在内存中则是连续分布的。

正是这样的主要区别，所以造成了tensor的一个高效性，那怎么理解这层高效性呢？假如我们去对一个二维矩阵进行转置操作，如果我们利用list去存这个二维矩阵，因为空间不连续，我们要么去改动地址中存储的内容，要么改动每个元素的映射地址。而如果我们使用tensor去存储这个二维矩阵，我们根本不用改动地址和存储的内容，只需要更改一下步长就可以了，例如原先是一个2×4的数组A=[[1,2,3,4],[5,6,7,8]]，我们将这个数组中的元素按顺序存储在一个连续的地址内1，2，3，4，5，6，7，8。当第一维度+1的时候，我们让步长+4，例如当前索引A[0,1]，现在索引A[1,1]，原先的目标是“2”，现在的目标是“6”，我们只需要在连续存储的那部分空间加上这个步长4，就可以找到改动后的目标，同理当第二维度+1时步长+1。所以当我们对二维数组进行转置的时候只需要改动维度对应的步长即可，根本不需要改动存储的结构，这就是Tensor的一个高效性体现。

除此之外，tensor可以和numpy.ndarray进行转换，通过这种方式来让tensor可以使用到大量的numpy中的高效运算：

b = a.numpy() #从tensor到ndarray
a = torch.from_numpy(a) #从ndarray到tensor

需要注意的是，转换之后两者共享内存，这句话的意思就是加入你把tensor a转换成了ndarray b，你改动b里面的元素，a中对应的元素也会改变。

1.2 Tensor数据类型

在讲解tensor操作之前，我觉得有必要先说一下一共有哪些数据类型，这些类型可以在创建tensor的时候，通过dtype来指定。
（1）torch.float32或torch.float：32位浮点数
（2）torch.float64或torch.double：64位浮点数
（3）torch.float16或torch.half：16位浮点数
（4）torch.int8：有符号的8位整数
（5）torch.uint8：无符号的8位整数
（6）torch.int16或torch.short：有符号的16位整数
（7）torch.int32或torch.int：有符号的32位整数
（8）torch.int64或torch.long：有符号的64位整数
我们可以在创建tensor的时候指定数据类型：

a = torch.ones(2,3,dtype=torch.double)

也可以查询tensor的数据类型：

a.dtype

设置数据类型：

a = torch.randn(10,2)
a = a.type(torch.short)

转换类型（创建时不设置类型默认为torch.float）：

a = torch.zeros(10,2).to(torch.double)

1.3 Tensor的一些常用操作

更加详细的操作可以去查阅PyTorch的文档：https://pytorch.org/docs/stable/torch.html，这里只介绍一些比较常见的用法。

随机创建一个tensor：

a = torch.rand(1,2)
a = torch.randn(1,2)

这两个语法都可以创建一个1*2大小的tensor但是是有区别的，torch.rand是从均匀分布中取值，torch.randn是从标准正态分布中取值。

其余创建方法：

a = torch.empty(1,2)
a = torch.ones(1,2)
a = torch.ones_like(b)#创建一个与b维度相同的tensor a，内容随机
a = torch.zeros(1,2)
a = torch.tensor([5.5,3])
a = torch.randn_like(b)

查看维度：

a.size() #返回一个torch.size对象

只有一个元素的tensor获取内容：

x = torch.rand(1)
y = x.item()

转置：

a = torch.randn(3,5)
b = a.t() #tensor a的转置
a = torch.ones(3,4,5)
b = a.transpose(0,2) #转置指定维度

改变形状：

x = torch.rand(4,4)
y = x.view(16) #转换成16维tensor
z = x.view(-1,8) #转换为2×8的tensor

数学操作：

z = x + y #加法
z = x + 2 #tensor x中每个元素加2
z = x*x #tensor x中每个元素平方
z = torch.sqrt(x) # tensor x中每个元素开平方
z = x.mean() # tensor x的平均值，z也是一个tensor
