1.重建二叉树


解题思路：前序序列中的第一个值为根节点的值，因此可以直接root = Treenode(preorder[0]) ，在从中序序列中查preorder[0]所处的位置，其下标记为a，则中序序列的0a-1个数属于左子树的中序，a+1到最后属于右子树的中序，而左子树的前序实际上是前序的第1a个数，右子树的前序是a+1到最后。故利用递归的思想即可解决本任务。
root = TreeNode(preorder[0])
a = inorder.index(root.val)
root.left = self.buildTree(preorder[1:a+1],inorder[0:a])
root.right = self.buildTree(preorder[a+1:],inorder[a+1:])
