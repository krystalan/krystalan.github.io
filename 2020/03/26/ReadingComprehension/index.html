3月30日随笔

分别在demo数据集（100条训练100条测试）上跑了一下BERT-base和RoBERTa_wwm_large_ext，效果如下：


上图是BERT，下图是RoBERTa。
RoBERTa_wwm_large_ext，在GTX 1080ti上只能设置Batch_size为1，训练2epoch，而BERT-base是Batch_size设置为4，训练2epoch。
猜测可能是由于batch-size设置为1导致效果过差orz。

于是我又去官网上看了一下各个模型的参数和在CMRC2018阅读理解上的对比：


于是总结出了：
（1）效果方面
BERT_wwm_ext_base < ALBERT_large < ALBERT_xlarge
RoBERTa_wwm_ext < ALBERT_xlarge < RoBERTa_wwm_large_ext
BERT_wwm_ext_base < RoBERTa_wwm_ext
（2）模型大小方面
ALBERT_large < ALBERT_xlarge < BERT_wwm_ext_base = RoBERTa_wwm_ext < RoBERTa_wwm_large_ext

所以打算用RoBERTa_wwm_ext和ALBERT_xlarge来做初步的训练。预计今夜开战：
